# 석사 과정생 교육 과정 (3개월 Condition)

> **목표: 논문을 읽고, 구현하고, 커스터마이징할 수 있는 수준**
> 신입생 12주 과정 이후 또는 기존 석사 과정생 대상

## Condition 제도

입학/합류 후 3개월 Condition 기간. 매월 월간 발표에서 평가.

```
[Month 1] 논문 읽기 + 재현 기초     → ★ 1차 월간 발표 (15분)
[Month 2] 논문 재현 + 커스터마이징   → ★ 2차 월간 발표 (20분)
[Month 3] 독립 연구 + 과제 기여     → ★ 3차 월간 발표 (25분+Q&A) = 최종 평가
```

## 월간 발표 핵심

| 발표 | 핵심 제출물 | 통과 |
|------|-----------|------|
| 1차 | 논문 3편 요약 + 1편 재현 + Ablation | 50점↑ |
| 2차 | 도메인 5편 서베이 + 1편 전체 재현 + 커스터마이징 3가지 | 50점↑ |
| 3차 | 독립 연구 (주제→실험→Ablation 5가지→논문 초고→과제 기여) | 50점↑ |

## Condition 최종 등급

| 등급 | 기준 | 결과 |
|------|------|------|
| S | 3회 발표 모두 70점↑ | 독립 연구자, 학회 투고 준비 |
| A | 3회 발표 모두 50점↑ | 과제에서 독립적으로 실험 담당 |
| B | 2회 이상 50점↑ | 선배 지도 하에 과제 참여 |
| C | 50점 미만 2회 이상 | 보충 후 재평가 (1개월 연장) |

## Condition 이후

매달 월간 발표는 계속 (정식 연구원 양식):
- 지난달 계획 대비 달성도
- 이번 달 실험 결과
- 이슈/논의 사항
- 다음 달 계획
