  - 너희가 투입될 과제

## 0-2. 국책과제 구조 — 연구실이 돌아가는 방식

```
정부(산업부/과기부)가 과제 공모
    ↓
연구실이 제안서 작성 → 선정
    ↓
연구비 수주 (장비, 인건비, 학회 출장 등)
    ↓
매년 연차보고서 제출 → 평가
    ↓
통과 → 다음 년도 / 실패 → 과제 중단
```

- 연구실 운영이 과제에 달려 있음
- 학생들이 직접 모델 개발, 데이터 수집, 실험, 논문 작성
- 여기서 배우는 기술 = 과제에 바로 쓰이는 기술

## 0-5. AI를 잘한다는 건

모델 이름 많이 아는 게 실력이 아니다.

> **문제를 보고 풀이 방법이 바로 떠오르는 것이 실력이다.**

```
문제가 뭐지? → 데이터를 어떻게 쓰지? → 전처리는? → 어떤 모델? → 평가는?
```

이 사고 흐름이 자동으로 되면, 뭘 시켜도 할 수 있다.
