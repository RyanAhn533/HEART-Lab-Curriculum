## 정답

<details>
<summary>정답 보기 (먼저 직접 풀어보세요!)</summary>

### 1. 쇼핑몰 리뷰 감정
→ 이진분류 / 텍스트 / 토크나이징, 패딩 / BERT Fine-tuning / F1, AUC

### 2. 안전모 실시간 감지
→ 객체탐지(실시간) / 영상(CCTV) / 프레임 추출, 라벨링(bbox) / YOLOv8 / mAP, Precision, Recall

### 3. 주가 예측
→ 분류(오를지/내릴지) 또는 회귀(얼마?) / 시계열 / 윈도우 슬라이딩, 정규화 / LSTM / Accuracy 또는 MSE

### 4. 운전자 감정 탐지
→ 다중분류 / 이미지+영상(얼굴) / 얼굴 crop(detection 먼저), resize, 정규화 / CNN+LSTM 또는 Video Transformer / F1 (macro)

### 5. 집값 예측
→ 회귀 / 정형(표) / 결측치 처리, 스케일링, (피처 선택) / XGBoost(데이터 적으면) 또는 MLP(많으면) / RMSE, R²

### 6. CT 종양 세그멘테이션
→ 세그멘테이션 / 의료 이미지(3D) / 정규화, resize, (증강) / U-Net / Dice Score, IoU

### 7. 메일 부서 분배
→ 다중분류(5클래스) / 텍스트 / 토크나이징, 패딩 / BERT Fine-tuning / F1 (macro), Accuracy

### 8. 폭력 장면 감지
→ 이진분류(폭력/비폭력) / 영상 / 프레임 샘플링, resize / CNN+LSTM 또는 3D CNN(C3D, SlowFast) / F1, AUC

### 9. 콜센터 감정 분류
→ 다중분류(3클래스) / 오디오 / mel-spectrogram 또는 MFCC, 노이즈 제거 / CNN(spectrogram 이미지로) 또는 wav2vec2 / F1 (macro)

### 10. 자율주행 도로 인식
→ 세그멘테이션 / 이미지(카메라) / resize, 정규화 / DeepLab 또는 U-Net / mIoU, Dice

### 11. 식물 병충해 진단
→ 다중분류 / 이미지(잎사귀) / resize, augmentation(데이터 적을 수 있음) / CNN + Transfer Learning(ResNet, EfficientNet) / Accuracy, F1

### 12. 논문 번역
→ 생성(번역) / 텍스트(영→한) / 토크나이징(BPE) / Transformer(mBART, NLLB) 또는 API(GPT, Claude) / BLEU

### 13. 장비 고장 예측
→ 이진분류(고장/정상) 또는 시계열 이상탐지 / 시계열(센서) / 윈도우, 정규화, (결측치) / LSTM 또는 Transformer / F1, Precision(false alarm 주의)

### 14. CCTV 인물 추적
→ 객체탐지 + Re-ID(재식별) / 영상 / 프레임 추출, 얼굴/사람 crop / YOLO(탐지) + Re-ID 모델(추적) / mAP, Rank-1 Accuracy

### 15. ECG 부정맥 탐지
→ 이진분류(또는 다중분류) / 시계열(1D 신호) / 정규화, 윈도우, 노이즈 필터링 / 1D-CNN 또는 LSTM / F1, Sensitivity, Specificity
